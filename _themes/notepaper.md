A highly customizable theme, designed exclusively for Pico.

Features:

* Widgets
* Blog-Style Front Page
* Sub-Navigation
* Navigation Arrows
* Built-in Disqus or Facebook Comments
* Custom Themes
* User CSS Overrides
* Basic Search Function
* Basic Tag Widgets
* Very customizable, with extensive Configuration Options
